## 📌 Sobre o projeto

Este é um desafio de código que classifica o nível de um herói com base na sua quantidade de experiência (XP). O programa utiliza variáveis, operadores, estrutura de decisão e saída formatada.

## 🧠 Lógica aplicada

O herói recebe um nome e um valor de XP. Em seguida, uma cadeia de `if/else` determina o nível conforme as faixas estabelecidas:

| Faixa de XP       | Nível       |
|-------------------|-------------|
| < 1.000           | Ferro       |
| 1.001 – 2.000     | Bronze      |
| 2.001 – 5.000     | Prata       |
| 5.001 – 7.000     | Ouro        |
| 7.001 – 8.000     | Platina     |
| 8.001 – 9.000     | Ascendente  |
| 9.001 – 10.000    | Imortal     |
| >= 10.001         | Radiante    |
